## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Using packages from shared

The packages/shared/src folder of this monorepo is symlinked to src/shared in this repo. This makes development easier, but one gotcha is that any new packages installed in packages/shared must also be installed in apps/backend.
